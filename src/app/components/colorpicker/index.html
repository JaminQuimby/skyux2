The colorpicker module allows users to use a button to select a color. When this attribute is placed on an (input or button) element and wrapped in a sky-colorpicker component, the Element and colorpicker are created. This attribute must be set to the instance of the sky-colorpicker. This directive will use NgModel for binding data. Will return an object as follows:
Hex: String
*Hex6 does not contain opacity.
hsva: Object
.hue: number
.saturation: number
.value: number
.alpha: number
rgba: Object
.red: number
.green: number
.blue: number
.alpha: number
rgbaText: string
*Formated for CSS: rgba(0,0,0,0)
hsla: Object
.hue: number
.saturation: number
.lightness: number
.alpha: number
cmyk: Object
.cyan: number
.magenta: number
.yellow: number
.key: number
 Specifies a array of colors that will load in the colorpicker. Specifies the initial color to be used, default #FFF. Specifies the value property on the provided input or button, accepts rgba or hex
