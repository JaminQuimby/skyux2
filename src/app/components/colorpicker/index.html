The colorpicker module allows users to use a button to select a color. When this attribute is placed on an (input or button) element and wrapped in a sky-colorpicker component, the Element and colorpicker are created. This attribute must be set to the instance of the sky-colorpicker. This directive will use NgModel for binding data. Specifies a array of colors that will load in the colorpicker. Specifies the initial color to be used, default #FFF. Specifies the format rgba or hex for the button value.
