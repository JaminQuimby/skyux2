The datepicker module contains directives for selecting dates. Specifies the selected date in the calendar, when not specified it is set to `new Date()`. Specifies a minimum selectable date in the calendar. Specifies a maximum selectable date in the calendar. Specifies the starting day of the week in the calendar, where 0 sets the starting day to Sunday. When a user selects a new date using the calendar, this event emits the selected date object.
